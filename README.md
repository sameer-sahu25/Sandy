# Sandhyarani Prusty — Creative Portfolio

A modern, high-performance creative portfolio showcasing product design, craft, photography, and sketches. Built with a focus on smooth interactions, meaningful storytelling, and a clean aesthetic.

## 🚀 Tech Stack

This project is built using the following technologies:

- **Frontend Framework**: [React](https://reactjs.org/) (v18)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features

- **Interactive UI**: Smooth scroll-triggered animations and cursor-following effects.
- **Dynamic Showcase**: Page-by-page portfolio rebuilding with real HTML/CSS.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **SPA Routing**: Seamless navigation with React Router and Vercel optimization.

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sameer-sahu25/Sandy.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
